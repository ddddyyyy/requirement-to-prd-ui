# PRD to Prototype Skill

## 简介

**PRD to Prototype** 是一个智能开发助手技能,旨在将PRD文档中的UI设计稿转化为可交互的网页原型。它支持从设计解析、组件构建到原型输出的完整工作流,帮助产品经理和开发者快速获得可演示的网页原型。

## 核心功能

- **🎨 网页原型生成**:根据PRD中的UI设计稿或用户提供的设计稿,自动生成可在浏览器中直接运行的交互式 HTML 原型。

## 快速开始

### 触发方式

在对话中提供包含UI设计稿的PRD文档,例如:
> "根据这个PRD文档生成网页原型"

或者使用以下命令触发:

| 命令 | 描述 |
| :--- | :--- |
| `/prototype-generate` | 根据 PRD 或 UI 设计稿生成网页原型 |

## 工作流程

本技能将PRD转化为可交互原型:

1. **原型生成**
   - 基于PRD中的UI设计部分。
   - 生成可交互的HTML原型文件(参考规范:`reference/UIPrototypeGen.MD`)。

## 输出产物

- **网页原型**:独立的HTML文件,包含CSS/JS,可直接在浏览器中打开预览交互效果。

## 依赖与参考

- **原型生成规范**:遵循 `reference/UIPrototypeGen.MD` 中的定义。

## 适用场景

- 产品经理快速获得可演示的网页原型。
- 独立开发者验证产品想法并快速获得交互原型。
- 团队内部进行产品评审前的原型制作。
